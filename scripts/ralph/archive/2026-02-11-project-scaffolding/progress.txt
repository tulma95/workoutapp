# Ralph Progress Log

## Codebase Patterns
- Project root is at /Users/rikuhonk/koodailu/treenisofta
- PRD and progress files are in scripts/ralph/
- Docker Compose uses postgres:latest with named pgdata volume
- Env vars: DATABASE_URL, JWT_SECRET, PORT, NODE_ENV
- Backend app is split: app.ts (Express app export) and index.ts (server startup) for testability
- db.ts exports a shared knex instance from DATABASE_URL (will be replaced with PrismaClient in this epic)
- Backend typecheck: `npx tsc --noEmit -p backend/tsconfig.json`
- Frontend typecheck: `npx tsc --noEmit -p frontend/tsconfig.app.json`
- Tests mock db.ts with vi.mock(); import module under test after mock setup
- Run backend tests: `npm test -w backend`
