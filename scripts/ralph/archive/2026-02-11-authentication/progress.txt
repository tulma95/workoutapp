# Ralph Progress Log

## Codebase Patterns
- Express 5 with TypeScript, CommonJS module output
- Config loads env vars with dotenv; import `config` from `./config` to validate at startup
- Prisma v7 with `@prisma/adapter-pg` adapter, client generated to `src/generated/prisma/`
- Prisma connection via `new PrismaPg({ connectionString: process.env.DATABASE_URL })`
- Zod v4: use `z.email()` for email validation
- Auth service uses bcrypt for password hashing, jsonwebtoken for JWT
- Test files live in `src/__tests__/` mirroring the src structure
- Use supertest for API route integration tests
- PostgreSQL 18 (latest): volume mount at `/var/lib/postgresql` not `/var/lib/postgresql/data`
